How to Use Kdenlive: A Beginner's Guide

This guide explains how to use Kdenlive for video editing, focusing on tasks like splitting/joining MP4 files, adding text (e.g., for chess move annotations), and exporting for YouTube. It's tailored for Pop!_OS users with NVIDIA hardware. Based on official documentation as of 2025.

Installation on Pop!_OS

Install via terminal: sudo apt install kdenlive. For NVIDIA acceleration, install sudo apt install nvidia-cuda-toolkit. Launch from the menu or kdenlive.

User Interface Overview

Kdenlive's UI includes the Menu Bar, Toolbars, Workspace panels (Project Bin, Monitors, Timeline), and Status Bar. Default Editing workspace:

  • Project Bin: Stores clips, images, audio; add via "Add Clip or Folder".
  • Clip Monitor: Views selected bin clips.
  • Project Monitor: Previews timeline with effects.
  • Timeline: Arrange clips on tracks (V1/V2 for video, audio tracks).
  • Timeline Toolbar: Tools like Select (S), Razor (X) for cutting, Spacer (M) for moving groups.
  • Status Bar: Shows hints, clip names, edit tools (e.g., razor for splitting).

Customize workspaces via Menu > View > Workspaces. Use keyboard shortcuts for efficiency (e.g., Space to play).

Creating a New Project

Create a project folder (e.g., chess-video/). Open Kdenlive, File > New, select folder. Set profile matching your clips (e.g., 1080p 30fps for MP4; auto-suggested). Save often: File > Save.

Importing Clips

In Project Bin, click Add Clip or Folder to import MP4 videos. Drag to timeline tracks. MP4 is fully supported; ensure profile matches for smooth editing.

Basic Editing: Splitting and Joining Clips

Splitting: Select Razor Tool (X), click on clip at cut point. Or position playhead, Shift+R.

Joining/Rearranging: Drag clips to align (snap enabled). Use Spacer Tool (M) to move groups without gaps. Insert mode shifts clips right.

Zoom timeline: Ctrl+Mousewheel. Preview in Project Monitor.

Adding Text (Titles)

Create titles for overlays like chess notations:

  • In Project Bin, right-click > Add Title Clip (or Alt+T).
  • In Titler window: Click to add text field, format (font, size, color).
  • Add shapes/images if needed (Alt+R for rectangle).
  • Position: Drag or X/Y coords; align with toolbar icons.
  • Animations: Apply timeline effects like Transform for fades/motion.
  • Save: Create Title; adds to Bin.
  • Drag to timeline above video track for overlay (alpha channel auto-composites).

Adding Effects and Transitions

View > Effects. Search/double-click to add to selected clip (e.g., RGB Adjustment). Adjust in Effects Stack; add keyframes for changes over time. For transitions: Overlap clips, click corner > Add Composition (e.g., Wipe).

Exporting (Rendering)

Project > Render (Ctrl+Enter):

  • Select YouTube preset (MP4, H.264/AAC).
  • For NVIDIA: Choose NVENC H264 preset under Hardware Accelerated.
  • Set output file, adjust speed/quality.
  • Render to File.

Enable parallel processing for faster renders on your hardware.

For more, check official docs. If issues with NVIDIA, ensure drivers are updated.
