VirtualBox + Windows 11 on Pop!_OS – Complete Setup Guide (2026)

This guide covers installing Windows 11 in VirtualBox on Pop!_OS, optimizing for performance (especially for light gaming like Clash of Clans via emulator), debloating, and installing Guest Additions for smooth experience. Your hardware (14th Gen i9, RTX 4060, 64 GB RAM) handles this easily.

1. Initial VM Creation & Windows 11 Installation

  1. Download official Windows 11 ISO from Microsoft (search "Download Windows 11 Disk Image (ISO)"). Choose Pro edition to match your host license if desired.
  2. In VirtualBox → Machine → New:
    • Name: e.g., "Win11-Gaming"
    • ISO Image: Select your Windows 11 ISO
    • Type: Microsoft Windows / Version: Windows 11 (64-bit)
  3. VirtualBox auto-detects Windows 11 needs → it enables:
    • EFI (UEFI) boot
    • TPM 2.0 (virtual)
    • Secure Boot
    → Keep these enabled (required for clean Win11 install).
  4. Hardware allocation (adjust based on your 64 GB RAM / i9):
    • Base Memory (RAM): 8–16 GB
    • Processors: 4–8 cores (enable PAE/NX + Nested VT-x/AMD-V if shown)
    • Hard disk: 80–128 GB dynamic VDI
  5. Finish → Start VM → Install Windows normally.
  6. During setup: "I don't have a product key" → continue (activates later if you input your OEM key, but optional for hobby use).
Tip: If install complains about hardware requirements despite TPM/Secure Boot → rare in recent VirtualBox versions, but you can bypass via registry edits (search "Windows 11 bypass TPM VirtualBox" if needed).

B. Inside Windows Guest

  1. Set Power Plan: Search "Power Plan" → Choose "High Performance".
  2. Update Windows (Settings → Windows Update) – do this before heavy tweaks.

3. Installing VirtualBox Guest Additions (for auto-resize, better graphics, clipboard, etc.)

  1. With VM running and Windows desktop loaded:
    • Go to VirtualBox menu (top of host window): Devices → Insert Guest Additions CD image
    • Nothing may "pop up" on host—that's normal. It mounts the ISO silently as a virtual CD.
  2. Inside Windows 11 guest:
    • Open File Explorer (Win + E or This PC).
    • Look under "This PC" for new CD/DVD drive (usually D: or E: — labeled "VirtualBox_GAs_..." or similar).
    • Double-click the drive to open it.
  3. In the CD contents folder:
    • Find and double-click VBoxWindowsAdditions.exe (or VBoxWindowsAdditions-amd64.exe for 64-bit — prefer this one).
    • If it doesn't start: Right-click → Run as administrator.
    • Follow installer wizard: Next → Install (allow network/driver changes if prompted).
    • Reboot VM when finished.
  4. After reboot:
    • Screen auto-resizes when dragging VM window.
    • Mouse integrates seamlessly (no capture needed).
    • Shared clipboard / drag-drop / shared folders work (enable in Devices menu if desired).
    • Graphics feel much smoother (great for BlueStacks / Clash of Clans).
If CD drive doesn't appear: VM Settings → Storage → Add empty optical drive if missing → Devices → Optical Drives → Choose disk file → browse to /usr/share/virtualbox/VBoxGuestAdditions.iso on Pop!_OS host.

4. Debloating Windows 11 (reduce CPU/RAM usage – great for VM)

  1. Download O&O ShutUp10++ (free): oo-software.com/en/shutup10
  2. Run it → Use "Recommended" preset (disables telemetry, OneDrive auto-start, Cortana, ads, etc.).
  3. Apply → Reboot. Expect lower idle RAM/CPU (e.g., 1–2 GB instead of 3–4 GB).
  4. Optional extra: Run Win11Debloat script from GitHub (Raphire/Win11Debloat) for deeper cleanup.

5. Setting Up Clash of Clans (via Android Emulator)

  1. Recommended: BlueStacks (best for Clash – macros, multi-instance support).
    • Download: bluestacks.com
    • Install in VM → Sign in Google → Search/install Clash of Clans.
    • In BlueStacks settings: Enable high FPS/graphics.
  2. Alternatives: LDPlayer (lighter) or MuMu Player.
  3. With Guest Additions + 3D accel + debloating → runs smoothly for your few minutes/day playstyle.

Final Tips

  • Take snapshots often (Machine → Take Snapshot) before big changes.
  • If performance still feels off → try VMware Workstation Pro (also free now) as alternative hypervisor.
  • Enjoy tinkering on Pop!_OS / Cosmic while keeping Windows handy!
