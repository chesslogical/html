How to Join 1.mp4, 2.mp4, 3.mp4, 4.mp4 into One File with FFmpeg

1. Open Terminal and go to your folder

cd /path/to/your/videos
(example: cd ~/Videos or cd ~/Desktop)

2. Create the list file (exact order)

printf "file '1.mp4'\nfile '2.mp4'\nfile '3.mp4'\nfile '4.mp4'\n" > list.txt
This creates list.txt containing: 
file '1.mp4'
file '2.mp4'
file '3.mp4'
file '4.mp4'

3. Join the files — lossless & instant (recommended)

ffmpeg -f concat -safe 0 -i list.txt -c copy merged.mp4
→ Creates merged.mp4 in the same folder (no quality loss, takes seconds)

If you get an error about mismatched codecs (rare)

ffmpeg -f concat -safe 0 -i list.txt -c:v libx264 -preset veryfast -crf 18 -c:a aac merged.mp4
This re-encodes quickly with almost zero visible quality loss.

Done!

Your final file is merged.mp4 (or whatever name you chose). You can delete list.txt afterward if you want.
Works perfectly on Fedora Workstation 43 (and any recent Linux). FFmpeg is pre-installed on Fedora.
