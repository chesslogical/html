Screen Recording and Video Editing on Pop!_OS Cosmic

This guide replaces Windows Snipping Tool (for video) and Clipchamp on Pop!_OS Cosmic for creating chess videos for YouTube. Focuses on lightweight, GPU-accelerated tools using your NVIDIA RTX 4060.

Screen Recording: Replacing Snipping Tool for Video

Pop!_OS Cosmic's built-in screenshot tool is for images; use third-party apps for video. These are Wayland-compatible and leverage NVIDIA encoding.

Recommended: GPU Screen Recorder

Lightweight, GPU-accelerated for NVIDIA. Uses NVENC for low CPU usage. Supports region selection for chess games in browsers.

  • Installation: 
    sudo apt update
sudo apt install flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub com.dec05eba.gpu_screen_recorder
  • Launch: flatpak run com.dec05eba.gpu_screen_recorder
  • Usage: Tray app; select "Region", drag over chess area, record (hotkey: Ctrl+Shift+Alt+R). Outputs MP4.

Alternative: Kooha

Simpler UI, hardware encoding support.

  • Installation: flatpak install flathub io.github.seadve.Kooha
  • Usage: Select region, record to MP4/WebM.

Video Editing: Replacing Clipchamp

Recommended: Kdenlive

Free, open-source editor for adding text, effects, etc. Supports NVIDIA acceleration.

  • Installation: sudo apt install kdenlive or Flatpak: flatpak install flathub org.kde.kdenlive
  • GPU Setup: Install sudo apt install nvidia-cuda-toolkit. In Kdenlive, enable GPU processing and NVENC rendering.
  • Usage: Import MP4, trim, add text/titles for chess moves, export with YouTube presets.

Alternative: Shotcut

Simpler than Kdenlive, with text/effects and NVIDIA support.

  • Installation: sudo apt install shotcut

Pro Alternative: DaVinci Resolve

Free version available, but may need tweaks on Cosmic (check System76 guide).

This setup matches your Windows workflow. Update NVIDIA drivers via Pop! Shop if issues arise.
