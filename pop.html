Pop!_OS Hardening Guide for Secure Web Browsing

1. Keep Your System Updated

  • Run sudo apt update && sudo apt upgrade -y regularly.
  • Enable automated security updates:
    • sudo apt install -y unattended-upgrades debconf-utils
    • echo 'unattended-upgrades unattended-upgrades/enable_auto_updates boolean true' | sudo debconf-set-selections
    • sudo dpkg-reconfigure -fnoninteractive unattended-upgrades

2. Full Disk Encryption

  • Ensure you chose LUKS encryption during installation.
  • Verify with lsblk -f that your root partition is encrypted.

3. Configure the Firewall

  • sudo apt install -y ufw
  • Set defaults:
    • sudo ufw default deny incoming
    • sudo ufw default allow outgoing
  • Enable UFW: sudo ufw --force enable
  • Check status: sudo ufw status verbose

4. Remove Unnecessary Services

  • Remove SSH server:
    • sudo apt purge -y openssh-server
    • sudo systemctl disable --now ssh
  • Disable Bluetooth and printing if unused:
    • sudo systemctl disable --now bluetooth
    • sudo systemctl disable --now cups

5. Application Sandboxing with Firejail

  • Install Firejail and profiles:
    • sudo apt install -y firejail firejail-profiles
  • Launch your browser sandboxed:
    • firejail firefox or firejail chromium

6. Enforce AppArmor Profiles

  • Install and enforce all profiles:
    • sudo apt install -y apparmor apparmor-profiles
    • sudo aa-enforce /etc/apparmor.d/*

7. Antivirus and Malware Protection (Not Recommended)

  • For a Linux desktop dedicated to web browsing, antivirus software offers negligible benefit and may introduce additional attack surface.
  • If you handle untrusted files or need to scan for cross-platform malware, consider offline scanning in a controlled environment rather than running an AV service continuously.

8. Audit Logging

  • Install auditd:
    • sudo apt install -y auditd
    • sudo systemctl enable --now auditd
  • Review logs with journalctl or ausearch.

9. Lock Root Account

  • Disable root login:
    • sudo passwd -l root

10. Additional Tips

  • Enable Secure Boot in your BIOS/UEFI.
  • Use strong, unique passwords and consider a password manager.
  • Back up important data regularly and verify backups.

Automated Hardening Script

Copy and paste into a terminal to apply these settings:

#!/bin/bash
set -eux

# Update and upgrade
sudo apt update && sudo apt upgrade -y

# Enable unattended security updates
sudo apt install -y unattended-upgrades debconf-utils
echo 'unattended-upgrades unattended-upgrades/enable_auto_updates boolean true' | sudo debconf-set-selections
sudo dpkg-reconfigure -fnoninteractive unattended-upgrades

# Configure UFW
sudo apt install -y ufw
sudo ufw default deny incoming
sudo ufw default allow outgoing
sudo ufw --force enable

# Remove SSH server
sudo apt purge -y openssh-server
sudo systemctl disable --now ssh

# Disable Bluetooth and printing
sudo systemctl disable --now bluetooth
sudo systemctl disable --now cups

# Install and enforce AppArmor
sudo apt install -y apparmor apparmor-profiles
sudo aa-enforce /etc/apparmor.d/*

# Install Firejail
sudo apt install -y firejail firejail-profiles

# Install and enable auditd
sudo apt install -y auditd
sudo systemctl enable --now auditd

# Lock root account
sudo passwd -l root

echo "Pop!_OS hardening complete."
