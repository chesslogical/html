Linux Chess YouTube Recording Guide

Perfect 1080p/4K chess videos with audio — no more GNOME recorder or Clipchamp nonsense.

1. Install OBS Studio (Flatpak — works on Fedora & Pop!_OS)

flatpak install flathub com.obsproject.Studio
flatpak run com.obsproject.Studio   # first launch

Pin it to your dock/favorites afterward.

2. Quick 5-Minute Chess Scene Setup

  1. Delete the default scene → create new scene called Chess Game
  2. Add these sources (exact order doesn’t matter, but drag to crop/position):
Source TypeWhat to Select / SettingsWhy
Window Capture (PipeWire) + → Window Capture → Method: PipeWire
Window: your Chrome/Firefox with Lichess or Chess.com
Crop: Alt+drag edges to show only the board 		Captures just the chess window, follows resizing
Audio Output Capture (PipeWire) + → Audio Output Capture → Device: Default Gets move sounds & any background music
Audio Input Capture + → Audio Input Capture → Device: Your microphone
(lower volume to ~−10 dB if it peaks red) 		Your commentary
(Optional) Text + → Text (GDI+) or Text (Pango) → write title or opponent name Game info overlay
(Optional) Video Capture Device Webcam → resize to corner for facecam Personal touch

3. Best Recording Settings (NVIDIA 4060 NVENC)

Settings → Output → Mode: Advanced

  • Recording Format: mp4
  • Encoder: NVIDIA NVENC H.264 (or AV1 if you want future-proof)
  • Rate Control: CQP
  • CQ Level: 15–20 (lower = better quality, bigger file)
  • Resolution: 1920×1080 or your monitor res
  • FPS: 60 or 30
  • Recording Path: ~/Videos/Chess

4. Hotkeys (optional but nice)

Settings → Hotkeys

  • Start Recording → F9
  • Stop Recording → F10

5. Workflow

  1. Open Lichess/Chess.com in Chrome (Proton VPN on — no problem)
  2. Launch OBS → make sure the window capture locked on the board
  3. Hit F9 → play & talk
  4. F10 when done → file appears in ~/Videos/Chess
  5. Quick trim (optional): flatpak install net.losslesscut.LosslessCut → open file → cut intro/outro → Save as (no re-encode)
  6. Drag final mp4 straight to YouTube upload page → 1080p/4K premium instantly

Extra Tools (all Flatpaks)

  • LosslessCut – instant lossless trimming: flatpak install net.losslesscut.LosslessCut
  • HandBrake – if you ever need smaller files: flatpak install fr.handbrake.ghb
